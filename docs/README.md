# Documentation (`docs/`)

This folder contains supporting documentation and files for the **Browser Forensics Collection Tool** project submitted for IT 360.

## Contents

### `final_report.pdf`
The official written report submitted as part of the project deliverables.  
It includes:

- Project introduction and purpose  
- Technical implementation details  
- Tool output examples and explanation  
- Integrity verification overview  
- Lessons learned and conclusion  

### `Tool Directory Tree.png`
An image showing an example directory structure produced by the tool after a successful evidence collection run.

### `cookies_4w5y56z.default.xlsx`  
### `history_4w5y56z.default.xlsx`
Readable spreadsheet versions of extracted browser artifacts used in the written report and video walkthrough.

These files demonstrate sample outputs generated by the tool and help illustrate how evidence is organized and processed.

---

The `docs/` folder contains all documentation necessary to understand, evaluate, and present the tool as part of the final project.
